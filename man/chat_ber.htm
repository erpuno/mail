BER/BERT

INTRO

The chat_ber is a WebSocket BER/BERT client interface and client socket protocol representation and protocol termination. At first to packet BER ASN.1 formatter is being applied, then if failed, fallback to BERT proceeds.

This module may refer to: chat, chat_client, chat_server.
